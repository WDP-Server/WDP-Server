# Commands Reference

**All commands organized by system**

## Economy Commands

### Shop
```
/shop
```
Open the shop menu.

### Balance
```
/sc balance
/skillcoins balance
```
Check your balance.

### Admin
```
/sc give <player> <coins|tokens> <amount>
/sc take <player> <coins|tokens> <amount>
/sc set <player> <coins|tokens> <amount>
/sc check <player>
```
