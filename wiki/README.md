## Quick Start

### New Players

1. Complete **Tutorial Quests** (/start) → Earn 1,415 ⛃
2. Open **Shop** (/shop) and buy gear
3. Level **Skills** (/skills) for passive income
4. Set **Homes** (/sethome) at your bases

### Earning Coins

| Activity | Earnings |
|----------|----------|
| Tutorial | 1,415 ⛃ |
| Daily Quests | 50-200 ⛃/day |
| Skill Levels | 10-50 ⛃/level |
| Selling Items | 30% of buy price |

### Key Commands

| Command | Purpose |
|---------|---------|
| `/shop` | Open shop menu |
| `/skills` | View skill levels |
| `/start` | Begin tutorial |
| `/quest` | Daily quests |
| `/home` | Teleport home |
| `/rtp` | Random teleport |
| `/base` | Base protection |
